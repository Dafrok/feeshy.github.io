密度无关像素是一种为了应对不同屏幕像素密度差异而提出的长度单位。Apple 为这个单位命名 pt ，Google 称 dp 。

这是一个介于物理长度与像素之间的概念，智能手表上 1 dp 的实际长度大约是现实中的 0.15 mm，智能手机的 1 dp 的一般在 0.15 ~ 0.2 mm 之间，PC 显示器上的 1 dp 大约是 0.2 ~ 0.3 mm —— 刚好反映了逐渐增加阅读距离，界面的物理尺寸也随之增加的规律。

网页设计中的 CSS 像素 px 并没有对实际像素还是密度无关像素做出区分，不过实际上，常见的浏览器都是按照密度无关像素的方式来处理 px 的。
