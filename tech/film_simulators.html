智能手机上的胶片模拟 App 很多，但是真的能让胶片玩家用过都说好的，并不是那么好找。

这里维护一个列表，收集一些能给胶片玩家带来“很像”、“好用”感觉的 App。

除非特别说明，上下文提到的所有“模拟”、“模仿”均不做区分地指 simulate，而不是 analog

  • Darkr (iOS) 是一个意图在屏幕上再现黑白胶片摄影主要流程的 App

  • Nebi ( iOS | Android ) 是一个专注于为照片添加颗粒、划痕、漏光的 App，但不同于其他使用算法生成颗粒的 App，这个 App 中所有的颗粒划痕都是开发者手工扫描的

  • Dev it ( iOS | Android ) 是一个内置了冲洗数据库的暗房计时器，为黑白手冲省去了查阅厂商文档的麻烦。Android 版的中文翻译由我维护，如有疏漏请向我反馈

  • Gudak Cam ( iOS | Android ) 是一个模拟柯达一次性相机的 App，不拍满24张不能回放，会随机加漏光。色彩确实有点柯达低端卷的味道，后来又增加了一款黑白卷的内购

  • Huji Cam ( iOS |Android ) 顾名思义，这个是模拟富士的版本，的确一股浓厚的魔术手味

  • 8mm (iOS) 是一个模拟 8mm 和 16mm 胶片录像机的 App，抖动，甚至可以生成配上放映机音效的默片

  • VHS Camcorder ( iOS | Android ) 是一个模拟 VHS 磁带录像机的 App，严格来说并不该出现在这里，但是管他呢，从效果上说，这个 App 跟上面提到的胶片模拟 App 异曲同工，都属于那种模仿的比较像的类型
