Web Safe Fonts 一般指在各个操作系统广泛预装，方便直接以CSS调用的字体。这个页面将会尝试调用常见的 Web 安全字体，测试你的操作系统和浏览器对其的支持情况。

你也可以把本页面当作一个字体挑选器来使用。

初步的候选字体列表来自 cssfontstack.com。最终的列表经我在 Apple 与 微软 的官网逐个确认手工筛选得到。

2019/06/03

说明

(Apple) Mac OS、iOS均有预装

(Mac) 仅Mac预装的字体 | (Mac D) Mac可用于文档显示的字体

(Win) 随Windows预装的字体

(Office) 随MS Office预装

1. 字体名称如若不加标签，则说明该字体在各个系统得到广泛的覆盖，推荐设计网页时首选无标签的字体

2. 同一段落显示的字体，多少有些相似性，适合把一种字体作为另一种的 fallback 使用

3. Android 在这里毫无存在感，是因为截止到目前 (Android P)，随原生 Android 预装的字体仅仅有 Droid Sans、Droid Serif、Droid Sans Mono 三种，且其中仅有 Droid Sans 支持中文字符，以至于毫无讨论的必要。
